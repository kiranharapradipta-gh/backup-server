# Backup Images Server

Backend server that receives images from Termux client and stores them into organized folders automatically.

## Features

- Receives image uploads via API
- Stores files by category/folder
- Simple REST API
- Lightweight Node.js backend

## Tech Stack

- Node.js
- Express.js
- Multer (file upload)
- File system storage

## API Endpoint

POST `/upload`
